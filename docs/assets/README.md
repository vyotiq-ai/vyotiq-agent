# Vyotiq-ai Assets

This directory contains visual assets for the project.

## Contents

- `logo.png` - Project logo (add your logo here)
- Screenshots and GIFs for documentation

## Logo Guidelines

Recommended logo specifications:
- **Size**: 512x512 pixels (minimum 120x120 for README)
- **Format**: PNG with transparent background
- **Style**: Clean, modern, developer-friendly

## Adding Screenshots

Screenshots should be:
- High resolution (Retina recommended)
- Dark theme preferred (matches app default)
- Cropped to relevant content
- Named descriptively (e.g., `chat-interface.png`, `editor-diff-view.png`)
